# 项目简介

## 核心功能
- 教师预约管理
- 分享与交流平台
- 公告资讯发布
- 个人信息管理（修改密码、聊天记录、评价教师、评价学生、投诉反馈）
- 用户管理
- 教师管理
- 课程分类管理
- 家教项目监控
- 评价系统维护
- 投诉处理
- 论坛分类管理
- 举报记录处理

## 适用场景
- 家教机构内部管理
- 教师和学生互动平台
- 教育资源分享和交流

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
